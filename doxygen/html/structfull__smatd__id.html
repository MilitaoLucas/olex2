Detailed Description

template<int base = 12>
struct full_smatd_id< base >

Full Id stores the matrix rotation part in first 18 bits, although for any crystallographic rotation matrix 13 bits are needed (nine bits for non zero matrix elements and 4 for signs - no more than 4 non-zero elements normally invloved, but holding the general rotaton with 64 bit number still allows huge fraction of 12 translaton: (64-18)/3 = 15 (+1): 2^14/12 = 1365 + sign... even base 256 allows translations +-63 cells
